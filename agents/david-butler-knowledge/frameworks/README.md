# frameworks/ — seeding convention

Holds psephological methodology, re-expressed in our own words with a citation.

**What belongs here:** swing math (uniform national swing, regional/proportional
swing), the swingometer model, projection/forecasting technique from partial results
and exit polls, race-call thresholds, seat-allocation formulas (FPTP, D'Hondt,
Sainte-Laguë, largest-remainder), turnout modeling.

**Each file needs:** the framework name, a plain-words explanation, a worked example,
a `Source:` citation (book/paper/author/year), a confidence tag, and a capture date.
Never copy text verbatim. After adding a file, bump `frameworks` in `INDEX.md`
`seed_counts`.
