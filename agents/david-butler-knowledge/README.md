# David Butler — Knowledge Base

Reference knowledge for the `david-butler` broadcast-psephology agent. This KB holds
durable methodology and benchmark data — re-expressed in our own words with citations,
never copied verbatim. Live, fast-changing facts (current results, live polls) are read
at runtime via WebFetch from `sources/official-sources.md`, not frozen here.

See `INDEX.md` for the manifest. Folders:
- `frameworks/` — swing, projection, seat-allocation, and turnout methodology
- `market-data/` — electoral results and polling benchmarks
- `persona/` — the cited homage profile (grounds the agent's voice; not a knowledge category)

This KB is a **scaffold** — folders and conventions are in place; populate as the team
captures durable knowledge.
